CONSOLIDATED CONFIGURATION BLOCK
Router Phase — Cisco C1111‑8P

This document contains the complete, OPSEC‑safe, authoritative configuration
for the router after the final freeze. All sensitive information (passwords,
secrets, keys) has been removed. All operationally relevant configuration is
preserved for academic, archival, and reproducibility purposes.
