ARCHITECTURE OVERVIEW — SWITCH PHASE
Cisco WS‑C3750X‑48PF‑E (Layer‑2 Core Switch)

1. ROLE IN NETWORK TOPOLOGY
---------------------------
The Catalyst 3750X functions as the Layer‑2 switching core for the homelab
infrastructure. It provides deterministic VLAN segmentation, hardened access
ports, and secure uplinks to the router and server. The switch enforces
zero‑trust boundaries and ensures that all inter‑VLAN routing occurs at the
router, not the switch.

The switch is responsible for:
- VLAN segmentation (10–100)
- Trunk uplinks to router and server
- Access ports for LAN and streaming devices
- Enforcement of L2 security controls (BPDU Guard, DHCP Snooping, etc.)
- Management-plane isolation via VLAN 10 (CORE‑10)

4. LAYER‑2 HARDENING CONTROLS
-----------------------------
The following controls are active and persistent:

- BPDU Guard (global)
- PortFast (access ports)
- DHCP Snooping (VLAN 10)
- Option 82 insertion enabled
- No trusted ports (correct for Phase 1)
- LLDP enabled
- CDP disabled
- HTTP/HTTPS server disabled
- Legacy services disabled:
    - finger
    - pad
    - tcp-small-servers
    - udp-small-servers
- Source routing disabled
- Service timestamps normalized
- Unused ports shutdown

These controls enforce a zero‑trust L2 environment and prevent common attack
vectors such as rogue DHCP servers, STP manipulation, and information leakage.
