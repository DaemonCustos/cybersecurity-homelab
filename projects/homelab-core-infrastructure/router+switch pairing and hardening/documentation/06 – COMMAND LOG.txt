COMMAND LOG
Switch + Router Pairing & Hardening Phase

This document records the operational commands used to validate and harden
the switch–router pairing. It is organized by device and by intent.

=====================================================================
1. ACCESS-SWITCH
---------------------------------------------------------------------
1.1 Interface and Trunk Validation
- show run interface GigabitEthernet1/0/1
- show interfaces trunk
- show spanning-tree interface GigabitEthernet1/0/1 detail
- show ip interface brief
- show vlan brief

1.2 Port Posture and Hardening
- show ip interface brief | include down
- show run | include interface GigabitEthernet1/0/
- show run | include shutdown

1.3 Management Plane and VTY ACL
- show run | section line vty
- show access-lists
- show run | include ip ssh
- show run | include login local

1.4 DHCP Snooping
- show ip dhcp snooping
- show ip dhcp snooping statistics
- show run | include dhcp snooping

1.5 LLDP Neighbor Validation
- show lldp neighbors
- show lldp neighbors detail

1.6 Logging and Services
- show logging
- show run | include logging
- show run | include http
- show run | include cdp
- show run | include lldp
- show run | include service

1.7 Save and State Verification
- write memory
- show startup-config | include version
- show version

=====================================================================
2. CORE-ROUTER
---------------------------------------------------------------------
2.1 Interface and Subinterface Validation
- show ip interface brief
- show run interface GigabitEthernet0/0/1
- show run interface GigabitEthernet0/0/0
- show run | include GigabitEthernet0/0/1.
- show run | include encapsulation dot1Q

2.2 VLAN Gateway and Reachability
- show ip route
- ping 10.0.10.2
- ping 10.0.20.1
- ping 10.0.70.1
- ping 10.0.100.1

2.3 ACL and Segmentation
- show access-lists
- show run | include ip access-group
- show run | section ip access-list

2.4 NAT Boundary
- show run | include ip nat
- show ip nat statistics
- show ip nat translations

2.5 DHCP Relay
- show run | include ip helper-address
- show ip dhcp server statistics

2.6 CoPP (Control Plane Policing)
- show policy-map control-plane input
- show run | section policy-map
- show run | section class-map

2.7 Logging and Management
- show logging
- show run | include logging
- show run | include ip ssh
- show run | section line vty

2.8 Save and State Verification
- write memory
- show startup-config | include version
- show version
