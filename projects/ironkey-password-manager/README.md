\# IronKey Offline Password Manager



\## Purpose

Build a secure, offline password management solution using an IronKey device and KeePassXC, with optional layered encryption and operational security controls.



\## Objectives

\- Create a portable, hardware-protected password vault

\- Implement a hardened KeePassXC configuration

\- Add optional intermediate encryption layers

\- Develop a shoulder-surfing resistant workflow

\- Create a system-scrubbing script for local machines

\- Document reproducible steps for internship credit



\## Architecture

\- IronKey (hardware encryption)

\- KeePassXC database stored on IronKey

\- Optional VeraCrypt or encrypted container layer

\- Local workstation with no persistent artifacts

\- Sanitized configs stored in this repo



\## Project Structure

\- diagrams/ — architecture diagrams, workflow charts

\- documentation/ — step-by-step build guides

\- configs/ — sanitized KeePassXC configs, IronKey notes

\- logs/ — sanitized logs or verification notes

\- scripts/ — scrubbing scripts or automation
