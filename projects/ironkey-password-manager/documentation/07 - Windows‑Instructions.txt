=== IronKey KeePassXC Portable – Windows Instructions ===

This IronKey contains a portable version of KeePassXC for Windows.
No installation is required, and no files are written to the host
system. All database files remain stored securely on the IronKey.

------------------------------------------------------------
1. Unlock the IronKey
------------------------------------------------------------
Use the IronKey unlocker application. Once unlocked, the secure
partition will appear as a drive letter in File Explorer.

Example:
KINGSTON (E:)

------------------------------------------------------------
2. IMPORTANT: Password Mode Requirement
------------------------------------------------------------
The IronKey D500S supports two password modes:

- Passphrase Mode
- Complex Mode

Windows supports both modes.

Linux supports ONLY Complex Mode.

If you intend to use this IronKey on Linux, the device MUST be
configured in Complex password mode. Passphrase mode will NOT work
on Linux under any circumstances due to a firmware limitation in
the IronKey Linux unlocker.

If the device is currently in Passphrase mode, switch it to Complex
mode using the IronKey control panel on Windows before attempting
to use it on Linux.

------------------------------------------------------------
3. Navigate to the Windows KeePassXC folder
------------------------------------------------------------
Open the following directory on the IronKey:

KeePassXC-Windows

Inside this folder you will find:

- KeePassXC.exe
- DIGEST file
- SOURCE.txt

------------------------------------------------------------
4. Run KeePassXC Portable
------------------------------------------------------------
Double-click:

KeePassXC.exe

KeePassXC will launch immediately without installation.

------------------------------------------------------------
5. Open your KeePassXC database
------------------------------------------------------------
Your database file is stored on the IronKey. Open it directly from
the device. No files are copied to the Windows host.

------------------------------------------------------------
6. Close KeePassXC when finished
------------------------------------------------------------
When you close KeePassXC:

- No configuration files are left behind
- No registry entries are created
- No temporary files remain on the host
- All data stays on the IronKey

This ensures a clean, zero-residue workflow.

------------------------------------------------------------
7. Notes
------------------------------------------------------------
- KeePassXC.exe is a portable application and does not require
  administrator privileges.
- The IronKey secure partition must remain unlocked while KeePassXC
  is running.
- Always close KeePassXC before unplugging the IronKey.
- If Linux compatibility is required, ensure the IronKey is set to
  Complex password mode before use.

------------------------------------------------------------
End of Windows Instructions
